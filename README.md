## Strict Super Potential

Strict Super Potential is a higher-threshold observation list built on top of Super Potential. It focuses on two profiles:

- **Strong confirmation**: price is close to, or has just moved above, the box-top area, but the breakout is not excessively stretched.
- **Calm setup**: short-term heat is lower, volume starts to activate, and MACD momentum improves.

Public explanation:

- close to the box-top area, or only slightly above it;
- volume starts to expand;
- MACD trend improves;
- box structure should not be too wide or too narrow;
- reduce mid-move and unstable chase samples.

## Swing Selected

Swing Selected is a swing-oriented observation strategy. It does not only look at the current day's percentage gain; it focuses more on whether momentum, volume, and box position remain balanced.

Public explanation:

- **Pullback watch**: the stock has not broken above the box top yet, but is already near the key area.
- **Breakout confirmation**: the stock has moved above the box top, but the breakout distance is still controlled.

## 严选超级潜力股

严选超级潜力股是在“超级潜力”基础上继续过滤出来的更高门槛样本。它主要关注两类股票：

- **强势确认型**：股价已经接近或刚越过箱体上轨，涨幅较强，但突破距离不能过远。
- **温和蓄势型**：涨幅不过热，量能开始活跃，MACD 趋势改善，适合继续观察。

公开口径可以理解为：

- 接近箱体上轨或刚突破不远；
- 量能开始放大；
- MACD 趋势向上；
- 箱体结构不能过宽或过窄；
- 主动减少中途追高和胜率不稳定的样本。

## 波段严选

波段严选更偏向“主线确认后的波段观察”，不会只看当天涨幅。它更重视量能、趋势、箱体位置和是否过热。

公开口径可以理解为：

- **回踩低吸型**：尚未突破箱顶，但已经接近关键区域；
- **突破确认型**：已经突破箱顶，但突破距离仍然可控。
