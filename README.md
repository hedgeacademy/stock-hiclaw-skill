# stock-hiclaw skill

`stock-hiclaw skill` 是 Stock Hiclaw 股票箱体突破研究系统的公开策略说明版本。

Stock Hiclaw 目前包含 4 个策略模块：

1. **今日突破**：已经满足箱体突破条件的强信号观察名单。
2. **超级潜力**：接近箱体上轨，或刚突破不远的未来观察名单。
3. **严选超级潜力股**：在超级潜力基础上继续提高门槛，重点保留强势确认型和温和蓄势型样本。
4. **波段严选**：偏向主线确认后的回踩低吸和突破不远样本，尽量避开过热追高。

本仓库公开展示其中两个策略的核心说明：

- 严选超级潜力股
- 波段严选

## 严选超级潜力股

严选超级潜力股是在“超级潜力”基础上继续过滤出来的更高门槛样本。它主要关注两类股票：

- **强势确认型**：股价已经接近或刚越过箱体上轨，涨幅较强，但突破距离不能过远。
- **温和蓄势型**：涨幅不过热，量能开始活跃，MACD 趋势改善，适合继续观察。

公开口径可以理解为：

- 接近箱体上轨或刚突破不远；
- 量能开始放大；
- MACD 趋势向上；
- 箱体结构不能过宽或过窄；
- 主动减少中途追高和胜率不稳定的样本。

## 波段严选

波段严选更偏向“主线确认后的波段观察”，不会只看当天涨幅。它更重视：

- 是否靠近箱体上轨；
- 涨幅是否处在相对可控区间；
- 量能是否温和放大，而不是极端放量；
- MACD 是否具备趋势确认；
- 长周期涨幅和振幅是否仍有空间；
- 是否避免已经过热的追高样本。

公开口径可以理解为：

- **回踩低吸型**：尚未突破箱顶，但已经接近关键区域；
- **突破确认型**：已经突破箱顶，但突破距离仍然可控。
